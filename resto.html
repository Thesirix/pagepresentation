Site Web et Application pour restaurant

Ce projet est un pack complet. C'est l'application d'un restaurant de pizza appairé à son site web dynamique.
On peut modifier le menu dans base de données et tout se met à jour dans l'application et le site web.

Détail technique : L'application est codé en python avec le framework kivy, les pizzas sont créées via un dictionnaire de 4 clés (nom,ingredient,prix,végétarienne) le contenu des dictionnaires est obtenue via le load d'un json crée avec une demande à l'API du site web. Le site web quand à lui est fabriqué grâce au framework django, les pages sont écrites en html et agencé en Css.
La base de donné peu être reçu ou exporté en format db en étant géré par un lecteur SQlite.

Technologies utilisées : Python, Kivy, Django,SQlite, Html, Css
