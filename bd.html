BackDoor

J’ai créé une Backdoor transférable d'un ordinateur à l'autre en utilisant les sockets résaux serveur/client

L’objectif ? Permettre la communication et le tranfert de données entre les deux machines via le protocole handshake.

Notamment utile pour faire de l'assistance à distance

Technologies utilisées : PYTHON

Github : https://github.com/Thesirix/BackDoor
